# Оптимизация системы активного шимминга МРТ

Проект посвящен разработке программного комплекса для расчета токов в шиммирующих катушках магнитно-резонансного томографа (МРТ). Цель проекта — минимизация неоднородности главного магнитного поля $B_0$ с использованием различных алгоритмов многомерной оптимизации.

## Описание

В реальности главное магнитное поле томографа имеет искажения. Для их устранения используется набор дополнительных (шиммирующих) катушек. Наша задача — подобрать такие токи $I_n$ для каждой катушки, чтобы их суммарное магнитное поле компенсировало исходную неоднородность. 

В проекте реализованы:
- Математическая модель магнитного поля (закон Био-Савара) с учетом пространственной геометрии 12 катушек.
- Ускорение расчетов за счет предрасчета единичных полей (принцип суперпозиции).
- Сравнительный анализ алгоритмов `scipy.optimize` (Nelder-Mead, L-BFGS-B, Differential Evolution, Dual Annealing).

## Структура проекта

* `src/` — исходный код движка (физика, константы, функции оптимизации и визуализации).
* `notebooks/Main_Experiment.ipynb` — главный Jupyter Notebook с экспериментами, анализом и графиками.
* `requirements.txt` — список зависимостей.

## Установка и запуск

1. Клонируйте репозиторий или скачайте папку с проектом.
2. Рекомендуется создать виртуальное окружение:
   ```bash
   python -m venv venv
   source venv/bin/activate  # Для Linux/Mac
   venv\Scripts\activate     # Для Windows
   ```
3. Установите необходимые библиотеки:
   ```bash
   pip install -r requirements.txt
   ```
4. Запустите Jupyter Notebook:
   ```bash
   jupyter notebook
   ```
5. В открывшемся окне браузера перейдите в папку `notebooks` и откройте файл `Main_Experiment.ipynb`.
