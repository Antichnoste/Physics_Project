# Оптимизация системы активного шимминга МРТ

Проект посвящен разработке программного комплекса для расчета токов в шиммирующих катушках магнитно-резонансного томографа (МРТ). Цель проекта — минимизация неоднородности главного магнитного поля $B_0$ с использованием различных алгоритмов многомерной оптимизации.

## Описание

## 1. Физика процесса
В магнитно-резонансной томографии (МРТ) качество получаемого изображения критически зависит от однородности главного магнитного поля ($B_0$, обычно 1.5 или 3.0 Тл). Любые искажения (неоднородности) приводят к быстрому расфазированию спинов протонов (уменьшение времени $T_2^*$), что вызывает размытие снимков и геометрические искажения. 

Неоднородность поля измеряется в миллионных долях (PPM — parts per million). Для качественной работы МРТ требуется неоднородность менее 1-5 PPM в рабочей сфере. 

**Активный шимминг** — это процесс компенсации этих искажений с помощью набора дополнительных электромагнитных катушек. Пропуская через них постоянные токи ($I_n$) различной величины и полярности, мы создаем компенсирующее магнитное поле, которое складывается с главным полем $B_0$ по принципу суперпозиции.

## 2. Математическая модель
Для расчета магнитного поля $\vec{B}$, создаваемого каждой шиммирующей катушкой в любой точке пространства $\vec{r}$, используется **закон Био-Савара-Лапласа**:

$$ d\vec{B} = \frac{\mu_0 I}{4\pi} \frac{d\vec{l} \times (\vec{r} - \vec{r}')}{|\vec{r} - \vec{r}'|^3} $$

Где $\mu_0$ — магнитная постоянная, $I$ — сила тока, $d\vec{l}$ — элемент контура катушки.
Для ускорения вычислений в программном продукте интеграл берется численно (путем разбиения кольца на $N$ сегментов), а поля вычисляются один раз для тока $1 А$ (формируется базис единичных полей).

## 3. Что и как мы исследуем
Мы исследуем эффективность алгоритмов многомерной оптимизации для подбора оптимальных токов $\vec{I} = \{I_1, I_2, ..., I_{12}\}$ в 12 шиммирующих катушках. 

**Целевая функция ($F$), которую минимизируют алгоритмы:**
$$ F(\vec{I}) = \frac{\sigma(B_{total})}{\langle B_{total} \rangle} \times 10^6 \quad \text{(в PPM)} $$
Где $\sigma$ — стандартное отклонение модуля суммарного магнитного поля в рабочей зоне, а $\langle B \rangle$ — его среднее значение.

В рамках проекта сравниваются градиентные методы (L-BFGS-B), симплекс-методы (Nelder-Mead) и глобальные стохастические алгоритмы (Differential Evolution, Dual Annealing) по критериям скорости сходимости и минимально достижимой неоднородности (PPM).

## 4. Что является продуктом
Продуктом является интерактивный программный комплекс (веб-приложение симулятор). Он позволяет:
1. Задавать произвольное расположение катушек в пространстве (симуляция сборки томографа).
2. Выбирать математический алгоритм оптимизации.
3. В реальном времени рассчитывать необходимые токи для компенсации искажений и визуализировать 3D-карту магнитного поля до и после применения шимминга.
В проекте реализованы:
- Математическая модель магнитного поля (закон Био-Савара) с учетом пространственной геометрии 12 катушек.
- Ускорение расчетов за счет предрасчета единичных полей (принцип суперпозиции).
- Сравнительный анализ алгоритмов `scipy.optimize` (Nelder-Mead, L-BFGS-B, Differential Evolution, Dual Annealing).

## Структура проекта

* `src/` — исходный код движка (физика, константы, функции оптимизации и визуализации).

## Установка и запуск

1. Клонируйте репозиторий или скачайте папку с проектом.
2. Рекомендуется создать виртуальное окружение:
   ```bash
   python -m venv venv
   source venv/bin/activate  # Для Linux/Mac
   venv\Scripts\activate     # Для Windows
   ```
3. Установите необходимые библиотеки:
   ```bash
   pip install -r requirements.txt
   ```
4. Запустите Jupyter Notebook:
   ```bash
   jupyter notebook
   ```
5. В открывшемся окне браузера перейдите в папку `notebooks` и откройте файл `Main_Experiment.ipynb`.
